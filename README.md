# 📊 Netflix Data Analysis & Preprocessing

## 📖 Overview

This project focuses on applying a complete data analysis pipeline on the Netflix dataset.
It covers data understanding, cleaning, exploratory data analysis (EDA), and feature engineering.

The goal is to transform raw data into meaningful insights.

---

## 🎯 Objectives

* Understand dataset structure
* Handle missing and inconsistent data
* Explore patterns and trends
* Create new features for better analysis

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 🔄 Workflow

Data Loading → Data Understanding → Data Cleaning → EDA → Feature Engineering

---

## 📊 Key Insights

* Movies are more common than TV Shows on Netflix
* Certain countries contribute more content
* Content production has increased over the years
* Some genres dominate the platform
